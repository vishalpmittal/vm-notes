# Glossary

| Term | Definition | Reference |
|---|---|---|
| Bias | How well a model fits the training data (high bias = bad train fit) | [ML Foundations](ml-foundations.md#bias-vs-variance--the-diagnostic-frame) |
| Classification | Supervised ML task with discrete labels as output (e.g., spam vs. not-spam) | [ML Foundations](ml-foundations.md#supervised-ml) |
| Clustering | Unsupervised task that groups similar objects into buckets | [ML Foundations](ml-foundations.md#unsupervised-ml) |
| Decision Boundary | Threshold (commonly 0.5) that converts a probability into a discrete class | [ML Foundations](ml-foundations.md#logistic-regression) |
| Dependent Feature | The output variable a model is trained to predict (the Y) | [ML Foundations](ml-foundations.md#supervised-ml) |
| Dimensionality Reduction | Compressing high-dim data into fewer dims while preserving structure (PCA, LDA) | [ML Foundations](ml-foundations.md#unsupervised-ml) |
| Generalized Model | Low bias + low variance — good performance on train and test | [ML Foundations](ml-foundations.md#bias-vs-variance--the-diagnostic-frame) |
| Hypothesis | The model's predicted relationship between inputs and output | [ML Foundations](ml-foundations.md#supervised-ml) |
| Independent Feature | An input to the model (the X) | [ML Foundations](ml-foundations.md#supervised-ml) |
| LDA | Linear Discriminant Analysis — supervised dimensionality reduction | [ML Foundations](ml-foundations.md#unsupervised-ml) |
| Logistic Regression | Linear model + sigmoid + decision boundary; used for binary classification | [ML Foundations](ml-foundations.md#logistic-regression) |
| Multicollinearity | When input features are highly correlated with each other; violates linear-regression assumption | [ML Foundations](ml-foundations.md#linear-regression--assumptions) |
| Overfitting | Low bias + high variance — model memorized training data, poor test performance | [ML Foundations](ml-foundations.md#bias-vs-variance--the-diagnostic-frame) |
| PCA | Principal Component Analysis — unsupervised dimensionality reduction | [ML Foundations](ml-foundations.md#unsupervised-ml) |
| Regression | Supervised ML task with a continuous numeric output | [ML Foundations](ml-foundations.md#supervised-ml) |
| Reinforcement ML | Paradigm where an agent learns through trial and reward signals | [ML Foundations](ml-foundations.md#reinforcement-ml) |
| Sigmoid | Function that squashes any real number into (0, 1); used in logistic regression | [ML Foundations](ml-foundations.md#logistic-regression) |
| Supervised ML | Training on labeled `(input, known output)` pairs to learn the mapping | [ML Foundations](ml-foundations.md#supervised-ml) |
| Underfitting | High bias + high variance — model can't even fit training data | [ML Foundations](ml-foundations.md#bias-vs-variance--the-diagnostic-frame) |
| Unsupervised ML | Training without labels to find structure (clustering, dimensionality reduction) | [ML Foundations](ml-foundations.md#unsupervised-ml) |
| Variance | How well training-data fit generalizes to unseen data (high variance = poor test fit) | [ML Foundations](ml-foundations.md#bias-vs-variance--the-diagnostic-frame) |
| VIF | Variance Inflation Factor — quantitative check for multicollinearity (>5 is a problem) | [ML Foundations](ml-foundations.md#linear-regression--assumptions) |
